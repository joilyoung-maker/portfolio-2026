# 웨이트 트레이닝 기록 앱 구현 계획 (Weight Training Tracker)

사용자가 헬스장에서 운동 정보(종목, 무게, 세트, 횟수)를 매일 직관적이고 빠르게 기록할 수 있는 미니멀리즘 스타일의 웹 애플리케이션입니다. React를 사용하여 현대적이고 모바일 친화적으로 구축합니다.

## User Review Required
> [!IMPORTANT]
> 본 계획은 **React**와 **Vite**를 사용하여 애플리케이션을 구축하는 것을 제안합니다. 이 방식은 모바일 웹 환경에서 빠르고 매끄러운 반응성을 제공합니다. 괜찮으신지 확인 부탁드립니다.

## 제안하는 기술 스택 및 아키텍처
*   **프레임워크:** React (Vite 기반, 빠른 빌드 및 로딩)
*   **스타일링:** 순수 CSS (Vanilla CSS)를 활용한 화이트/블랙 모노톤 미니멀리즘 디자인
*   **데이터 저장:** 브라우저 `localStorage` 활용 (오프라인 상태 및 앱 재실행 시 유지)
*   **날짜 처리:** 네이티브 JavaScript `Date` 객체

## 데이터 구조 (LocalStorage)
```javascript
// 1. workouts: 날짜별 운동 기록 매핑
{
  "2026-04-26": [
    { id: "uuid-1", name: "벤치 프레스", weight: 60, sets: 4, reps: 10 },
    { id: "uuid-2", name: "스쿼트", weight: 80, sets: 3, reps: 12 }
  ]
}

// 2. recentExercises: 최근 사용한 운동 종목 리스트 (빠른 입력용)
["벤치 프레스", "스쿼트", "데드리프트", "풀업", "랫풀다운"]
```

### 2단계: 핵심 상태 및 로직 구현 (데이터)
*   `localStorage`에서 데이터를 불러오고 저장하는 훅(Hook)을 작성합니다.
*   현재 선택된 날짜 상태를 관리하고, 날짜 좌우 이동 버튼을 통해 데이터를 동적으로 전환하도록 구현합니다.

### 3단계: 화면(View) 및 입력 최적화 구현
*   **AddWorkoutForm:** `기구명, 무게, 세트, 횟수`를 입력받습니다.
    *   입력 칸 상단에 최근 사용한 운동 목록을 "추천 태그"로 띄워, 클릭 한 번으로 기구명이 입력되도록 만듭니다.
*   **WorkoutList:** 입력된 리스트를 깔끔한 카드 형태로 출력하고 삭제 기능을 추가합니다.
