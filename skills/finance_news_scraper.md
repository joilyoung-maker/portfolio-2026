# 금융 뉴스 스크래핑 Skill

## 목적
네이버 금융의 최신 뉴스를 추출하여 분석 가능한 데이터로 정제합니다. 모든 관련 로직은 TypeScript로 구현됩니다.

## 지침
1. `src/news_scraper.ts` 파일을 생성하거나 수정하여 스크래핑 로직을 작성한다.
2. `axios` 또는 `puppeteer`를 사용하여 네이버 금융 뉴스 페이지에 접속한다.
3. 뉴스 제목, 언론사, 게시 시간을 추출한다.
4. 추출된 데이터를 `data/news_list.json` 또는 CSV 형식으로 저장한다.

## 성공 기준
- `src/news_scraper.ts` 파일이 존재하고 에러 없이 실행됨.
- `data/` 디렉토리에 결과 데이터가 저장됨.
- 모든 코드는 TypeScript 타입을 엄격히 준수함.
